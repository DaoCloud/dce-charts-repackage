# OpenBao

[![Pipeline](https://gitlab.com/gitlab-org/cloud-native/charts/openbao/badges/main/pipeline.svg)](#)

:warning: This project is in early development stage and is not yet suitable
for production use.

This chart installs [OpenBao](https://openbao.org/) for GitLab, which is required to enable the [GitLab Secrets Manager](https://docs.gitlab.com/ci/secrets/secrets_manager/).
